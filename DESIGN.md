# DESIGN.md — Video Learner GUI

> 本文档描述 Electron GUI（`electron/src/renderer/index.html`）的界面目标、屏幕清单、UI 状态与交互流程。
> 设计令牌、组件约定见 [`docs/reference/design-system.md`](docs/reference/design-system.md)。

---

## 一、界面目标

| 目标 | 说明 |
|---|---|
| **零学习曲线** | 粘贴 URL → 点运行，无需了解后台流水线细节 |
| **进度透明** | 8 步流水线实时可见，失败步骤可单步重试 |
| **内容优先** | 文章/摘要占最大面积；视频播放、字幕、元信息在右侧按需查看 |
| **视觉风格** | Swiss Minimal — 黑白灰、直角、无装饰、8px 栅格（见 design-system.md） |

---

## 二、整体布局

```
┌──────────────────────────────────────────────────────────────────────┐
│  侧栏 220px  │        主内容区 flex:1         │▕│  视频面板 400px     │
│              │                                │ │                    │
│  ┌──────────┐│  工具栏（打开/删除/管理/中止）  │ │  [播放] [信息]      │
│  │ 任务历史 ││  ──────────────────────────── │ │                    │
│  │  列表    ││  [文章] [摘要] 内容标签栏      │ │  视频容器           │
│  │          ││  进度条（处理中时显示）         │ │  播放控制栏         │
│  │          ││  ──────────────────────────── │ │  ━━━ 分割线 ━━━    │
│  │          ││                               │ │  字幕列表           │
│  │          ││  文章 / 摘要 内容区             │ │                    │
│  └──────────┘│                               │ │  （或信息面板）     │
└──────────────────────────────────────────────────────────────────────┘
```

**尺寸约束：**
- 侧栏：固定 220px
- 视频面板：默认 400px，拖拽范围 260–720px
- 视频容器高度：默认 225px，随面板宽度保持视频宽高比，拖拽范围 160–480px
- 两处分割线均可拖拽（横向：`#resizer`；纵向：`#videoResizer`）

## 三、屏幕清单与 UI 状态

### 3.1 空状态（无任务选中）

**触发条件：** 首次启动，或历史列表为空，或用户未点击任何任务

| 区域 | 显示内容 |
|---|---|
| 侧栏历史列表 | 空列表（无任务时显示"No history"提示） |
| 工具栏 | 隐藏（`hidden`） |
| 文章面板 | `#emptyState`：「输入 YouTube 链接开始」 |
| 摘要面板 | `#summaryEmptyState`：「选择已完成的任务查看摘要」 |
| 视频面板 | `#videoEmpty`：▶ 暂无视频 |

**进入方式：** 点击「+ New」打开新建弹窗

---

### 3.2 任务选中（文章/摘要内容展示）

**触发条件：** 点击侧栏任意任务条目

| 区域 | 显示内容 |
|---|---|
| 侧栏条目 | 高亮（左边框 2px accent，背景 bg-alt） |
| 工具栏 | 显示「打开 / 删除 / 管理」；运行中时加「中止」；已中止时加「继续」 |
| 文章面板 | 渲染 Markdown 文章；未生成时显示「文章尚未生成（或读取失败）」 |
| 摘要面板 | 渲染 Markdown 摘要；未生成时显示「总结尚未生成（或读取失败）」 |
| 视频容器 | 加载本地视频文件（视频/音频均可）；无媒体文件时显示 `#videoEmpty` |
| 字幕区 | 有字幕轨时显示字幕列表，无字幕时隐藏 |
| 信息面板 | 显示标题、创作者、URL、时长、输出语言、关注点、各步骤状态、创建/更新时间 |

**内容加载顺序：**
1. `GET /tasks/:id` → 更新信息面板 + 工具栏状态
2. `GET /tasks/:id/content/article` + `/summary` → 渲染 Markdown
3. `GET /tasks/:id/media` → 加载视频/音频
4. `GET /tasks/:id/subtitles` → 渲染字幕列表

---

### 3.3 任务运行中

**附加于「任务选中」状态之上，进度条可见**

| UI 元素 | 状态 |
|---|---|
| `#progressSection` | 显示（`.hidden` 移除） |
| 进度条 `#progressFill` | 按完成步骤数/总步骤数更新宽度 |
| 进度步骤点 `.progress-step` | `active`（脉冲动画）/ `done`（实心点）/ 默认（空心点） |
| 工具栏「中止」按钮 | 显示（红色危险按钮） |
| 侧栏条目状态点 | 黄色（running） |
| 信息面板步骤 pill | `active`（旋转图标 ◐）/ `done`（✓）/ `error`（✗ 可点击重试） |

**8 个步骤（按流水线顺序）：**

| `data-step` | 显示名 | 说明 |
|---|---|---|
| `fetch` | 获取 | yt-dlp 获取视频元信息 |
| `video` | 视频 | 下载视频文件 |
| `audio` | 音频 | 下载音频文件 |
| `subs` | 字幕 | 下载字幕轨 |
| `vtt2md` | 转文案 | VTT 字幕转 Markdown |
| `md2vtt` | 字幕生成 | Markdown 转回带时间戳 VTT（用于对齐） |
| `article` | 文章 | LLM 生成文章 |
| `summary` | 摘要 | LLM 生成摘要 |

---

### 3.4 视频播放器

**位于视频面板「播放」tab 下**

| 元素 | 行为 |
|---|---|
| 视频容器 | 黑色背景；视频元素 `max-width/max-height: 100%`；无视频时显示 `#videoEmpty` |
| 播放/暂停按钮 | 点击切换；播放时显示 ⏸，暂停时显示 ▶；`aria-label` 同步更新 |
| 停止按钮 | 暂停并跳回 0:00 |
| 进度条 `range` | 拖拽跳转；`timeupdate` 事件实时更新 |
| 时间显示 | 格式 `m:ss / m:ss`（超过 1 小时则 `h:mm:ss`） |
| 分割线 `#videoResizer` | 竖向拖拽，调整视频容器与字幕列表的高度分配 |

**字幕区（`#subtitleModule`，有字幕时显示）：**
- 字幕轨 ≤ 2 条：语言切换按钮（中文 / English）
- 字幕轨 > 2 条：降级为 `<select>` 下拉
- 字幕条目点击：跳转到对应时间点并播放
- 当前播放位置的字幕条目高亮（黑底白字），自动滚入视图
- 「画面内字幕」复选框：切换视频内嵌 TextTrack 字幕显示

---

### 3.5 信息面板

**位于视频面板「信息」tab 下**

| 字段组 | 字段 |
|---|---|
| 视频信息 | 标题、创作者、URL、时长 |
| 任务配置 | 输出语言、关注点 |
| 处理进度 | 8 步 status pill（同运行中状态） |
| 时间 | 创建时间、更新时间 |

**错误步骤交互：**  
status pill 状态为 `error` 时变为可点击（`.clickable`），hover 显示「重试」提示，点击打开重试弹窗。

## 四、弹窗清单

### 4.1 新建任务弹窗（`#newTaskModal`，`data-mode="new"`）

**打开方式：** 点击侧栏「+ New」或工具栏「打开」（无任务选中时）

| 字段 | 说明 |
|---|---|
| URL | YouTube 链接，必填 |
| 关注点 | 自由文本，如「技术架构」「主要论点」，可选 |
| 资源类型 | 单选：视频 / 音频 / 仅字幕（对应 `mode: video/audio/transcript`） |

**底部：**
- 「运行」主按钮 → 提交任务，进入运行状态
- status pill 列表实时更新步骤状态
- 日志面板（`#modalLogs`）显示 SSE 流式日志

**关闭：** × 按钮或点击蒙版外区域

---

### 4.2 管理任务弹窗（同 `#newTaskModal`，`data-mode="edit"`）

**打开方式：** 工具栏「管理」按钮

与新建弹窗共用同一 modal 容器，区别：
- 标题变为「管理任务」
- URL / 关注点字段只读
- 「运行」按钮隐藏；任务运行中时「■ 停止任务」按钮显示
- 步骤 pill 展示当前各步状态，失败步骤可通过 pill 触发重试
- 日志面板展示历史日志

---

### 4.3 删除确认弹窗（`#confirmDeleteModal`）

**打开方式：** 工具栏「删除」按钮

自定义下拉选择器，三种删除模式：

| 模式 | 说明 |
|---|---|
| 硬删除（默认） | 删除数据库记录 + 磁盘文件，**不可恢复** |
| 仅删记录 | 仅删数据库记录，保留磁盘文件 |
| 标记删除 | 列表中隐藏，不删记录与文件 |

操作按钮：取消 / 删除（红色危险按钮）

---

### 4.4 重试步骤弹窗（`#retryConfirmModal`）

**打开方式：** 点击信息面板中状态为 `error` 的 status pill

**两阶段 UI：**

1. **确认阶段**（`#retryModalPhaseConfirm`）
   - 标题：「是否重试 {步骤名}？」
   - 「自动执行后续步骤？」复选框
   - 取消 / 确认按钮

2. **运行阶段**（`#retryModalPhaseRunning`）
   - 「重试中…」提示
   - SSE 日志流面板

---

### 4.5 重置步骤弹窗（`#resetPopup`）

> ⚠️ **当前状态：** DOM 已实现（Swiss Minimal 风格），JavaScript 连接尚未完成，功能暂未激活。

预期行为：确认后重置指定步骤状态，允许重新执行该步骤（不同于「重试」——重试保留失败记录，重置清除状态）。

## 五、侧栏交互

### 历史列表（`#historyList`）

- 按 `created_at` 倒序排列，最多显示 200 条
- 每条目：标题（截断，单行），日期，状态点
- 状态点颜色：🟡 running · 🟢 completed · 🔴 failed · 🟠 aborted · ⚫ unknown
- 点击条目：`selectTask(id)` 加载该任务内容
- 活跃条目：左边框 2px accent

### 搜索（`#sidebarSearch`）

实时过滤历史列表，匹配标题（大小写不敏感），清空搜索词恢复全列表。

### SSE 实时更新

任务状态变化通过 SSE 推送，侧栏历史列表防抖刷新（400ms）。

## 六、工具栏按钮状态规则

| 按钮 | 显示条件 |
|---|---|
| 打开 | 始终显示（无任务时打开新建弹窗；有任务时打开外部链接/文件） |
| 删除 | 始终显示 |
| 管理 | 始终显示 |
| 中止 | 仅当前任务 `status === 'running'` |
| 继续 | 仅当前任务 `status === 'aborted'` |

工具栏整体在空状态下隐藏（`hidden`），选中任务后显示。

---

## 七、键盘 / 焦点

- 新建弹窗打开后焦点自动落在 URL 输入框
- 管理弹窗打开后焦点落在关注点输入框
- Tab strip 使用 `role="tablist/tab/tabpanel"` + `aria-selected`，支持屏幕阅读器
- 视频控制按钮有 `aria-label`，播放/暂停时动态更新
