神经网络从零开始：构建 GPT 的完整指南

本视频由 Andrej Karpathy 主讲，系统讲解了从标量反向传播到完整 GPT 模型的实现过程。核心思路是通过"从零构建"来真正理解每个组件的工作原理。

核心模块

Karpathy 首先演示了如何手工实现自动微分引擎 micrograd，仅用约 100 行 Python 代码即可完成标量级别的反向传播，帮助理解梯度计算的本质。

在此基础上逐步扩展到 makemore 字符语言模型，引入 Bigram、MLP、BatchNorm 等模块，最终构建完整的 Transformer 架构并实现一个可运行的 GPT。

关键洞见

视频强调实践优先：每个概念都配有可运行的代码，并通过对比 PyTorch 内置模块来验证手写实现的正确性。
