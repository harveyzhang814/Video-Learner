这是 Andrej Karpathy 关于大语言模型的一次深度技术讲解，覆盖了从预训练到 RLHF 的完整流程。整个视频长度 15 分钟，内容密度极高，几乎没有水话。

如果你只听一次入门讲座，这一个就够了。

一、预训练（Pretraining）

预训练是 LLM 训练的第一个阶段，目标是让模型学会"世界知识"。核心方法是 next-token prediction。

数据规模

  • 训练语料：通常 10T+ tokens，来自互联网爬取 + 书籍 + 代码
  • 计算资源：数千张 H100 GPU，运行 2-3 个月
  • 成本：单次训练 $10M+

输出形态

预训练完成后得到的是一个 base model。它已经掌握了大量知识，但还不擅长"对话"——它只会"接下文"。

二、监督式微调（SFT）

SFT 阶段教会模型用助手式语气回应。人工写一批高质量的「问题 → 答案」配对，让模型在这些数据上继续训练。

SFT 不是教模型新知识，而是教它新格式。

三、RLHF（人类反馈强化学习）

这一步是 ChatGPT 真正"好用"的关键。流程分三步：

  • 训练一个 reward model 来评估回答质量
  • 用 PPO 算法让 LLM 最大化 reward
  • 持续收集人类反馈，迭代优化

四、未来方向

Karpathy 提到几个值得关注的方向：多模态、长上下文、agentic 能力，以及更高效的训练方法。
