# Summary

## TL;DR
Gemini CLI 是 Google 推出的终端智能代理工具，可通过自然语言指令完成文件处理、代码开发、数据分析等复杂任务，其开源性质和强大的工具生态系统使其成为提升开发效率的利器。

## Outline
1. Gemini CLI 简介与核心特点
2. 四大应用场景：开发、内容创作、数据分析、学习助手
3. 提示词处理流程与工作原理
4. 终端代理相比 Web 界面的核心优势
5. 内置工具与扩展生态

## Key Points
- **开源免费**：Gemini CLI 完全开源，由社区驱动开发，免费版本提供慷慨的请求配额
- **多场景应用**：支持软件开发、播客转短视频、CSV 数据处理、学习笔记整理等场景
- **工作原理**：提示词 → Gemini 模型推理 → 工具调用 → 迭代优化，形成完整任务闭环
- **终端优势**：直接访问文件系统、调用本地工具、自动化脚本、扩展生态
- **内置工具**：文件系统操作、网页搜索、Shell 命令执行、记忆功能

## Terms/Entities
- **Gemini CLI**: Google 推出的终端智能代理工具，基于 Gemini 大语言模型
- **MCP (Model Context Protocol)**: 模型上下文协议，用于扩展 Gemini CLI 功能
- **终端代理**: 运行在命令行界面的 AI 助手，可直接操作本地文件和应用
